¿Qué ofrece el proyecto a desarrollar?

Escalabilidad

SGDI JyE ofrece un crecimiento a medida que los requerimientos del cliente cambien o se añadan.

Interfaz Simple

Siempre se mantendrá una interfaz simple para la gestión de los datos, muy útil dependiendo de qué clase de usuario lo use.

Gestión de Datos

Una gestión de datos segura basada en las tecnologías seleccionadas para el desarrollo del proyecto.
