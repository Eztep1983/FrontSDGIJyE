¿Que ofrece el proyecto a desarrollar?

Escalabilidad

SGDI JyE ofrece un crecimiento a medida que los requerimientos del cliente cambien o se añadan.

Interfaz Simple

Algo caracteristico, siempre se manejara una interfaz simple para la gestion de los datos, muy util dependiendo que clase de usuario lo use.

Gestion de datos

Una gestion de datos de una manera segura en base a las tecnologias que se seleccionaron para el desarrollo del proyecyo.
